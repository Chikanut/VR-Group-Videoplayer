# EXE icon

Put Windows executable icon here:

- File name: `app.ico`
- Path used by builder: `App/assets/icons/app.ico`
- Recommended source size before converting to `.ico`: **1024x1024 px** (square)
- The `.ico` file should include multiple embedded sizes: `16, 24, 32, 48, 64, 128, 256`.

`build_windows_exe.bat` automatically uses this icon if file exists.
