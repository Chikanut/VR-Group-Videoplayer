# Підготовка відео

## Ціль

Підготувати навчальні відео так, щоб Quest player міг їх знайти і відкрити без додаткових налаштувань.

## Обов'язковий шлях зберігання

Скопіюйте кожне відео уроку в:

`/sdcard/Movies/`

Цей шлях важливий. Панель керування надсилає лише ім'я файлу, а player шукає цей файл саме в `/sdcard/Movies/`.

## Правило для імен файлів

У панелі керування використовуйте тільки ім'я файлу.

Приклади:

- правильно: `lesson01.mp4`
- правильно: `history_360.mp4`
- неправильно: `/sdcard/Movies/lesson01.mp4`

## Підтримувані режими перегляду

- `2D` або flat mode для звичайного відео на віртуальному екрані
- `360` mode для equirectangular 360-відео

## Рекомендований процес підготовки

1. Визначте фінальні імена файлів до копіювання.
2. Скопіюйте однакові імена файлів на всі гарнітури.
3. Відкрийте settings у панелі керування.
4. Додайте кожен урок як required video з цим ім'ям файлу.
5. Виберіть правильний video type для кожного уроку.

## Корисні поради

- використовуйте короткі та зрозумілі назви
- не перейменовуйте файли після того, як уже налаштували урок
- переконайтеся, що набір контенту однаковий на всіх гарнітурах
- протестуйте хоча б одне відео до початку заняття

## Очікуваний результат

Коли панель керування перевіряє пристрої, налаштоване ім'я файлу має показуватися як доступне на кожній гарнітурі.
