# Мережа та усунення проблем

## Ціль

Вирішити найпоширеніші причини, через які пристрої не з'являються або відео не запускається.

## Базові мережеві правила

- панель керування і всі гарнітури мають бути в одній локальній мережі
- мережа повинна дозволяти пристроям напряму бачити одне одного
- guest Wi-Fi або client isolation можуть блокувати пошук пристроїв

## Якщо гарнітури не знаходяться

Перевіряйте по черзі:

1. Чи відкритий player app на гарнітурі.
2. Чи гарнітура і панель керування підключені до однієї Wi-Fi або LAN мережі.
3. Чи параметр subnet у панелі відповідає вашій мережі.
4. Чи роутер не ізолює бездротових клієнтів.
5. Чи Windows firewall не блокує роботу, якщо пошук все ще не працює.

## Якщо гарнітура показується offline

- переконайтеся, що гарнітура все ще має доступ до мережі
- тримайте player app відкритим на передньому плані
- перевірте, що пристрій не перейшов у сон
- дочекайтеся наступного циклу сканування і оновіть dashboard

## Якщо відтворення не стартує

Перевірте:

1. що гарнітура online
2. що ім'я файлу уроку існує на цій гарнітурі
3. що відео скопійоване в `/sdcard/Movies/`
4. що вибраний режим відповідає типу контенту

## Якщо відео виглядає відсутнім

- порівняйте налаштоване ім'я файлу з реальним ім'ям без помилок
- не використовуйте шляхи до папок у панелі керування
- переконайтеся, що той самий файл є на всіх гарнітурах

## Якщо Windows не відкриває панель керування

- спробуйте відкрити `http://localhost:8000` вручну
- перезапустіть EXE-файл
- переконайтеся, що порт `8000` не зайнятий іншою програмою

## Розширене технічне посилання

Для низькорівневих деталей команд та endpoint дивіться [Player API reference](../../PlayerAPI.md).
