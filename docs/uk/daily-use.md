# Щоденне використання

## Ціль

Провести звичайне заняття через панель керування.

## Перед уроком

1. Увімкніть усі гарнітури.
2. Відкрийте player app на кожній гарнітурі.
3. Переконайтеся, що всі пристрої підключені до однієї мережі.
4. Відкрийте панель керування.
5. Переконайтеся, що кожен пристрій показується як online.

## Запуск уроку

1. Відкрийте dashboard.
2. Оберіть навчальне відео, яке потрібно запустити.
3. Перевірте, що потрібне ім'я файлу доступне на підключених гарнітурах.
4. Запустіть відтворення для всіх пристроїв або для однієї обраної гарнітури.

## Під час відтворення

Через панель керування можна:

- відтворювати
- ставити на паузу
- зупиняти
- перезапускати поточне відео
- центрирувати вид
- задавати global volume
- окремо змінювати гучність однієї гарнітури

## Recenter

Використовуйте recenter, якщо учні більше не дивляться в правильному напрямку. Це особливо корисно для 360-контенту або після того, як гарнітуру переміщували.

## Після уроку

1. Зупиніть відтворення.
2. Завершіть сесію або залиште player готовим для наступної групи.
3. Поставте гарнітури на зарядку, якщо батарея низька.
4. Оновлюйте відео лише якщо імена файлів залишаються узгодженими з налаштуваннями панелі.

## Очікуваний результат

Викладач або адміністратор лабораторії має мати можливість керувати всім заняттям без ручного налаштування кожної гарнітури окремо.
