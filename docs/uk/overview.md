# Огляд

## Призначення

VR Group Videoplayer - це система для класу або навчальної лабораторії, яка дозволяє запускати один і той самий навчальний відеоконтент на кількох гарнітурах Meta Quest з одного пристрою викладача.

Система орієнтована на школи, лабораторії, музеї та тренінгові простори, де потрібен зрозумілий процес налаштування без додаткової розробки.

## Що вміє система

- знаходити VR-плеєри в локальній мережі
- показувати, які гарнітури зараз онлайн
- зберігати список навчальних відео за іменем файлу
- запускати відтворення на одній гарнітурі або на всій групі
- ставити на паузу, зупиняти, перезапускати, центрирувати та змінювати гучність
- працювати і з плоским 2D-відео, і з 360-відео

## Основні компоненти

### 1. Quest Player

Встановлюється на кожну гарнітуру Meta Quest.

Роль:

- відтворює відео, що збережені на гарнітурі
- відправляє статус назад у панель керування
- приймає команди відтворення по локальній мережі

Публічний файл релізу:

- `VRClassroomPlayer-Quest.apk`

### 2. Панель керування

Встановлюється на один пристрій викладача.

Роль:

- знаходить доступні Quest player у мережі
- показує поточний стан гарнітур
- зберігає список обов'язкових відео для уроку
- надсилає команди відтворення та гучності

Публічні файли релізу:

- `VRClassroomControlPanel-Windows.exe`
- `VRClassroomControlPanel-Android.apk`

## Типовий сценарій використання

1. Встановити player на кожну гарнітуру.
2. Скопіювати навчальні відео на кожну гарнітуру.
3. Встановити панель керування на Windows ПК або Android-планшет викладача.
4. Підключити всі пристрої до однієї мережі.
5. Відкрити панель керування і дочекатися появи пристроїв.
6. Обрати відео уроку та запустити його для всієї групи.

## Підтримуваний сценарій

- гарнітури Meta Quest для відтворення
- Windows ПК для керування
- Android телефон або планшет для керування
- локальна Wi-Fi або LAN мережа, де пристрої можуть напряму бачити одне одного

## Важливі обмеження

- відео мають уже бути на кожній гарнітурі
- панель керування працює лише з іменем файлу
- відео на гарнітурі слід зберігати в `/sdcard/Movies/`
- система розрахована на локальну мережу, а не на хмарний стримінг
