# Налаштування панелі керування

## Ціль

Встановити та відкрити панель керування викладача на Windows або Android.

## Оберіть платформу

- використовуйте Windows, якщо потрібен більший екран і звичний desktop-сценарій
- використовуйте Android, якщо потрібен мобільний телефон або планшет

## Варіант A: Windows

### Передумови

- Windows ПК або ноутбук
- `VRClassroomControlPanel-Windows.exe` зі сторінки [GitHub Releases](https://github.com/Chikanut/VR-Group-Videoplayer/releases/latest)
- доступ до тієї ж Wi-Fi або LAN мережі, що й у гарнітур

### Кроки

1. Завантажте `VRClassroomControlPanel-Windows.exe`.
2. Запустіть файл на ПК викладача.
3. Якщо Windows покаже попередження безпеки, дозвольте запуск, якщо джерело релізу правильне.
4. Дочекайтеся відкриття вікна застосунку або сторінки в браузері.
5. Переконайтеся, що панель доступна за адресою `http://localhost:8000` на цьому комп'ютері.

### Очікуваний результат

Ви повинні побачити dashboard панелі керування, з якого пізніше можна буде знаходити Quest player у мережі.

## Варіант B: Android

### Передумови

- Android телефон або планшет
- `VRClassroomControlPanel-Android.apk` зі сторінки [GitHub Releases](https://github.com/Chikanut/VR-Group-Videoplayer/releases/latest)
- дозвіл на встановлення APK-файлів на пристрої

### Кроки

1. Завантажте `VRClassroomControlPanel-Android.apk`.
2. Встановіть APK на Android-пристрій.
3. Дозвольте встановлення з обраного джерела, якщо Android цього попросить.
4. Відкрийте застосунок.
5. Дочекайтеся завантаження вбудованого інтерфейсу панелі.

### Очікуваний результат

Ви повинні побачити ту саму панель керування, що і на Windows, але всередині Android-застосунку.

## Перші дії після запуску

Після відкриття панелі:

1. Відкрийте Settings.
2. Додайте навчальні відео за іменем файлу.
3. За потреби перевірте battery threshold, scan interval і network subnet.
4. Поверніться на dashboard і дочекайтеся появи пристроїв.
